**ONLINE MODE**

1.Prerequiresites:

-To co-op with the other, 2 devices must be in the same network layer

&emsp;&emsp;LAN/Wi-Fi: in the same router Wi-Fi

&emsp;&emsp;Internet: using tool allowing creating virtural LAN such as ZeroTier, RadminVPN, Tailscale,…

*Note*:If host is in PC (Windows), make sure opening port 5555 on Windows Firewall

2.How to play

&emsp;&emsp;In PC/Android 1, Multiplayer ➔ Host Game ➔ set name and choose color
&emsp;&emsp;In the other, Multiplayer ➔ Join Game ➔ choose the room with IP’s PC/Android 1
